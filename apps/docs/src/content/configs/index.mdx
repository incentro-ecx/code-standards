# Configs

The following shared configurations can be used:

| Config                             | Description                                           | Changelog                                   |
| ---------------------------------- | ----------------------------------------------------- | ------------------------------------------- |
| [CSpell](./configs/cspell)         | A shared CSpell configuration for spell checking.     | [Changelog](./configs/cspell/changelog)     |
| [ESLint](./configs/eslint)         | Shared ESLint configurations for linting files.       | [Changelog](./configs/eslint/changelog)     |
| [Prettier](./configs/prettier)     | A shared Prettier configuration for formatting files. | [Changelog](./configs/prettier/changelog)   |
| [Typescript](./configs/typescript) | Shared Typescript configurations.                     | [Changelog](./configs/typescript/changelog) |
